==============
pySCTranscform
==============

SCTransform for Python - interfaces with `ScanPy <https://scanpy.readthedocs.io/en/stable/>`_

* Support for `batch_var` missing

See `demo <notebooks/demo.ipynb>`_ for a comparison and `PBMC3k <https://github.com/saketkc/pySCTransform/blob/glmgp/notebooks/pbmc3k_umap.ipynb>`_ for comparison with log normalization approach.
